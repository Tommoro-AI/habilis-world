# 하빌리스 콘솔이란 

<br />
<div style="background:#f3f4f6;border:1px solid #33bfc1ff;padding:12px 14px;color:#000; font-size:16px;border-radius:10px;margin:12px 0;line-height:1.6;">  🌏️ <strong>  하빌리스 콘솔</strong>은 <br />휴머노이드 로봇 데이터를 수집하고, 변환하고, 학습하고, 현실에서 실행하기까지 전 과정을 견고한 파이프라인으로 연결하여 제공하는 휴머노이드 학습용 소프트웨어 생태계입니다.🌏️<br />
</div><br />

<strong>  “로봇을 학습시키는 일”</strong> 은 더 이상 연구실의 복잡한 스크립트들에 국한되지 않습니다.<br />휴머노이드 로봇으로 <strong>  데이터를 수집하고, 학습 자산으로 가공하고, AI를 훈련시키며, 다시 현실 세계에서 실행되기까지 
</strong> 하빌리스 콘솔은 흩어져 있던 전 과정을 하나로 엮어, <strong> 로봇 학습의 시작과 끝을 아우르는</strong> 통합 생태계를 제공합니다 <br /><br />

하빌리스 콘솔은 <strong> 다양한 로봇을 붙이고, 다양한 장치를 붙이고, 다양한 환경에서</strong>  반복 실험을 할 수 있도록 만들어졌습니다. 우리는 지금 곁에있는 로봇은 물론 앞으로 만나게 될 새로운 로봇까지 모두 함께 할 수 있는 확장가능한 공존을 지향합니다. <br /><br />

하빌리스 콘솔은 로봇 학습에서 가장 인기있는 포맷 중 하나인 <strong>  LeRobot 데이터 포맷</strong> 으로 변환기능을 제공합니다. 통일된 데이터 형식은 실험의 재현성과 공유 가능성을 높이고, 팀과 커뮤니티의 속도를 빨라지게 합니다. 데이터가 통일되는 순간, 성장이 시작됩니다.<br /><br />

하빌리스 콘솔의 또다른 매력은 사용 경험에 있습니다.<br />
로봇을 잘 모르는 사람도, AI 학습을 잘 모르는 사람도, 날것의 코딩을 다룰 필요 없이 화면에서 로봇을 연결하고 제어하고 데이터를 기록할 수 있습니다. <strong> 버튼 하나로 기록을 시작하고, 세션을 관리하고, 수집물을 확인하고, 필요한 변환과 큐레이션까지 이어지는 흐름을 느껴보세요.</strong> 이는 사용자의 실수를 줄이고, 반복을 쉽게 만들고, 팀의 확장을 수월하게 합니다. <br />
경험이 없어도 누구나 원하는 꿈에 집중할 수 있는 것이 하빌리스 콘솔이 추구하는 가치입니다.<br /><br />

우리는 이런 약속을 합니다.<br />로봇 학습의 여정이 지식의 깊이나 경험의 유무에 가로막히지 않도록—데이터에서 모델로, 모델에서 현실로 이어지는 길을 단단하게 만들고,어떤 로봇이든, 누구든 쉽게, 그리고 팀이 함께 빠르게 달릴 수 있는 생태계를 만들것입니다.<br />
현장의 움직임과 의도를, 학습 가능한 미래의 능력으로 바꾸는 도구.<br />그것이 저희가 제공하고싶은 휴머노이드 학습용 소프트웨어 생태계입니다.<br />
