# 2.2 다중 로봇 네트워크 추가
하나의 콘솔에서 여러 대의 로봇을 관리해야 할 경우, 새로운 네트워크를 추가하여 각각의 연결 정보를 독립적으로 관리할 수 있습니다.

## Step 1. 추가 네트워크 생성
1. 상단 메뉴의 로고 또는 홈 아이콘을 클릭하여 메인 대시보드로 이동합니다.

2. 화면 중앙 또는 상단에 위치한 **[Add Robot]** 버튼을 클릭합니다.
![console_config_add_robot](./image/2.1_console_new_add_robot.png)

3. 버튼을 클릭하면 목록에 새로운 Network 카드가 생성됩니다.

## Step 2. 추가 로봇 정보 입력
새로 생성된 네트워크 카드를 클릭하여 진입하면, [2.1 신규 디바이스 등록](2.1_new_device.md)와 동일하게 해당 로봇의 네트워크 이름과 IP 주소를 입력하고 저장합니다.

![console_config_add_robot2](./image/2.1_console_new_add_robot_click.png)
    
:::tip[참고]

각 네트워크는 서로 다른 IP 주소를 가질 수 있으며, 로봇마다 개별적으로 연결 상태를 확인할 수 있습니다.

사용하지 않는 네트워크는 설정 화면 하단의 **[Delete Network]** 버튼을 통해 삭제할 수 있습니다. 
:::
