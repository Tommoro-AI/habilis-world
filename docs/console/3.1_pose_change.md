# 3.1 로봇 자세 변경

Control 메뉴에서는 3D 프리뷰를 활용하여 실제 로봇 구동 전, 가상 모델로 안전하게 초기 자세를 설정할 수 있습니다.

## Step 1. Control 메뉴 접속
1. 좌측 메뉴 바에서 **[Control]** 탭을 클릭하여 제어 인터페이스 화면으로 진입합니다.
![console_robot_control_main](./image/3.1_console_robot_control_main.png)
    

## Step 2. 자세 설정 및 동기화
1. 화면의 각 관절별 스크롤바를 움직여 원하는 포즈를 만듭니다.

2. **[Sync]** 버튼을 누르면 **실제 로봇의 현재 자세**가 가상 모델에 동기화됩니다.
![console_robot_pose_sync.png](./image/3.1_console_robot_pose_sync.png)

## Step 3. 실제 로봇 적용
1. **[Apply]** 버튼을 누르면 **설정 자세가 실제 로봇에 적용**됩니다.
![console_robot_pose_apply.png](./image/3.1_console_robot_pose_apply.png)
    
:::danger[위험]
**[Apply] 버튼 클릭 시 실제 로봇이 즉시 구동됩니다!**
- 로봇의 가동 범위 내에 사람이나 장애물이 없는지 반드시 확인하십시오.
- 예기치 못한 급격한 움직임을 방지하기 위해, 초기 조정 시에는 스크롤을 조금씩 움직여 미세하게 조절할 것을 강력히 권장합니다.
![console_robot_pose_applyEX.png](./image/3.1_console_robot_pose_applyEX.png)
:::
