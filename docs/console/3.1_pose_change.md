# 3.1 로봇 자세 변경

**Control** 메뉴에서는 3D 프리뷰를 활용하여 실제 로봇을 구동하기 전, 안전하게 초기 자세를 설정하고 미세 조정할 수 있습니다.

1. 좌측 메뉴 바에서 **Control** 탭을 클릭하여 로봇 제어 인터페이스 화면에 진입합니다.
    
    [이미지]
    

1. 각 관절별로 스크롤 바를 조절하여 원하는 수치를 조정하고, **Sync** 버튼을 클릭하여 3D 프리뷰 상에서 변경된 자세를 확인합니다.

[이미지]

1. 3D 모델의 자세가 안전하다고 판단되면, **Apply** 버튼을 눌러 실제 로봇에 관절 값을 반영합니다.

<aside>
⚠️

> **[ 주의 ] 안전 주의사항**
> 
> - **Apply** 버튼을 누르는 즉시 실제 로봇이 구동되므로, 주변에 장애물이 없는지 반드시 확인하십시오.
> - 초기 조정 시에는 스크롤을 미세하게 조절하는 것을 권장합니다.
</aside>
