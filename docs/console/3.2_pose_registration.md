# 3.2 로봇 자세 등록
설정이 완료된 로봇의 자세는 시스템에 프리셋으로 등록하여 이후에도 재사용이 가능합니다.

## Step 1. 자세 업데이트 창 열기
1. 로봇의 자세가 확정되었다면, 화면 상단의 <strong> [Custom Pose]</strong> 버튼을 클릭합니다.
![console_robot_pose_update](./image/그림31.png)

## Step 2. 포즈 이름 입력 및 저장    
1. <strong>Update Pose</strong>팝업창이 나타나면, 해당 자세를 구분하기 쉬운 이름을 입력합니다.
    - 예: `Initial Pose`, `Ready Pose`, `Task 01`
2. 입력 후 우측의 **[추가]** 버튼을 클릭하여 저장합니다.
![console_robot_pose_update](./image/그림32.png)
