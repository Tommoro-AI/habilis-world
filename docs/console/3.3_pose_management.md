# 3.3 로봇 자세 프리셋 관리

- Update Pose 창 기능은 다음과 같습니다.

![console_robot_pose_list](./image/3.3_console_robot_pose_update_list.png)

1. 포즈 갱신: **[Update]** 버튼을 눌러 현재 로봇의 자세로 해당 포즈 정보를 갱신합니다.
2. 포즈 구동: **[Recall]** 버튼을 눌러 저장된 해당 포즈로 로봇을 즉시 구동시킵니다.
    
    

:::danger[위험]
-  **[Recall]** 버튼을 누르는 즉시 실제 로봇이 구동되므로, 주변에 장애물이 없는지 반드시 확인하십시오.
:::

3. 포즈 삭제: **[Delete]** 버튼을 눌러 해당 포즈를 목록에서 영구적으로 삭제합니다.
