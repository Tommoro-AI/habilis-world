# 3.3 로봇 자세 프리셋 관리


`Update Pose` 창에서는 목록의 버튼을 클릭하여 로봇의 자세를 관리하거나 저장된 위치로 구동할 수 있습니다.

![console_robot_pose_list](./image/3.3_console_robot_pose_update_list.png)

<div class="field-table">
  <div class="field-row">
    <div class="field-label">Update</div>
    <div class="field-desc">현재 로봇의 실시간 자세를 해당 항목에 <strong>갱신</strong>하여 저장합니다.</div>
  </div>

  <div class="field-row">
    <div class="field-label">Recall</div>
    <div class="field-desc">저장된 포즈 값에 맞춰 로봇을 즉시 <strong>구동</strong>시킵니다.</div>
  </div>

  <div class="field-row">
    <div class="field-label">Delete</div>
    <div class="field-desc">해당 포즈 데이터를 목록에서 영구적으로 <strong>삭제</strong>합니다.</div>
  </div>
</div>

:::danger[위험]
- **[Recall]** 버튼 클릭 시 실제 로봇이 즉시 움직이므로, 구동 범위 내에 장애물이나 사람이 없는지 반드시 확인하십시오.
:::
