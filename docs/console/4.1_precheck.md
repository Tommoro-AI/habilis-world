# 4.1 사전 점검
로봇의 포즈를 기반으로 **ROS2 Bag** 형식의 데이터를 수집하기 전, 시스템 상태를 최종 점검하는 단계입니다.

Record 메뉴에 접속하여 아래 사항을 반드시 점검합니다.
- 정면(Front) 및 측면(Side) 영상이 지연 없이 송출되는지 확인합니다.
- 영상 하단 캡션 정보가 현재 로봇의 실제 포즈 및 상태 데이터와 일치해야 합니다.
- 데이터 저장용 하드웨어의 전원 및 네트워크 연결 안정성을 확인합니다.

:::warning[경고]
**1. 영상 하단 캡션과 영상이 모두 표시되지 않음**  
- 로봇 제어 확장팩과 dataset-manager 확장팩이 정상 실행 중인지 확인합니다.
- Config 페이지에서 `ROBOT IP` 주소가 실제 로봇 IP와 일치하는지 재확인합니다.
<br />
**2. 영상 하단 캡션은 나오지만 영상이 출력되지 않음**  
- Config 페이지로 이동하여 디바이스 연결을 해제한 후 재연결합니다.
:::
