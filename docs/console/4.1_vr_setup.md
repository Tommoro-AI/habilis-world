# 4.1 VR 기기 설정
:::info[정보] 
이 챕터는 <strong>메타퀘스트3(Meta Quest 3) 헤드셋을 사용하여 데이터를 수집하는 사용자</strong>에게만 해당합니다. 

VR 기기를 사용하지 않는 경우 다음 단계로 넘어가셔도 좋습니다. 
:::

메타퀘스트3를 활용하여 로봇을 원격 제어(Teleoperation)하고 데이터를 수집하기 위한 환경 설정 단계입니다.

## Step 1. 메타퀘스트3 환경세팅
원활한 데이터 수집을 위해 메타퀘스트3 내부 설정을 아래와 같이 변경합니다.

1. **개발자 모드 활성화**: [메뉴 > 기기 > 헤드셋 정보 > 개발자 모드]를 ON으로 설정합니다.

2. **절전 모드 해제**: 수집 중 화면이 꺼지는 것을 방지하기 위해 시간을 최대치로 설정합니다.
    - [설정 > 일반 > 전원] → 디스플레이 끄기(4시간), 절전 모드(4시간)
![예시 이미지](./image/4.1_(5).jpg)

4. **핸드 트래킹 활성화**: [설정 > 움직임 트래킹] → 손 제스처 및 신체 동작 트래킹 활성화
![예시 이미지](./image/4.1_(6).jpg)

## Step 2. 작업 공간 설정

데이터 수집 중 컨트롤러나 손이 인식 범위를 벗어나지 않도록 충분한 <strong>경계(Boundary)</strong>를 설정합니다.
![예시 이미지](./image/4.1_(7).jpg)

:::warning[주의]
경계 밖으로 나갈 경우 텔레오퍼레이션 연결이 즉시 끊길 수 있습니다.
:::

Step 3. 하빌리스 콘솔 접속 및 연결
1. **IP 확인**: PC의 하빌리스 콘솔에서 VR Page address:를 확인합니다
![예시 이미지](./image/4.1_(8).png)

2. **브라우저 접속**: 메타퀘스트3 하단 바의 <strong>Browser(보라색 아이콘)</strong>를 실행한 후, 확인한 콘솔 주소를 입력합니다.
![예시 이미지](./image/4.1_(9).jpg)
![예시 이미지](./image/4.1_(10).jpg)

3. **인증 및 확인**: 
    - 브라우저 좌측 상단의 [Trust Cert] 버튼을 클릭합니다.
    - ROS 아이콘이 <strong>Connected(초록색)</strong>로 변경되는지 확인합니다.
![예시 이미지](./image/4.1_(11).jpg)

4. **VR 진입**: 브라우저 우측 상단의 [Enter VR] 버튼을 눌러 제어 화면으로 진입합니다.

## Step 4. VR 사용 종료
손바닥이 본인을 향하게 둔 상태에서 엄지손가락과 새끼손가락를 맞닿게 하면 VR 모드가 종료됩니다.

:::warning[주의]
- PC와 메타퀘스트3는 반드시 동일한 와이파이 네트워크에 연결되어 있어야 합니다.
- VR 진입 후 화면이 멈추거나 반응이 없다면, 메타퀘스트 브라우저를 새로고침한 뒤 다시 <strong>[Enter VR]</strong>을 눌러주세요. 
:::
