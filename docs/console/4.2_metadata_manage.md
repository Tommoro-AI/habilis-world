# 4.2.1 메타정보 작성
<br />

**Task Information** 섹션에 현재 수행할 작업의 상세정보를 작성합니다. 이 정보는 추후 데이터 관리 및 학습 시 식별자로 활용됩니다.

<div> 
  <img src="./image/4.2_task_information.png" width="680"/>
</div>

<div class="field-table">
  <div class="field-row">
    <div class="field-label">Task Name</div>
    <div class="field-desc">수행할 <strong> 작업의 이름</strong>을 입력합니다.</div>
  </div>

  <div class="field-row">
    <div class="field-label">Tags</div>
    <div class="field-desc">각 에피소드의 특성이나 내용을 구분하기 위한 <strong> 태그</strong>를 입력합니다. (선택)</div>
  </div>

  <div class="field-row">
    <div class="field-label">Episode duration</div>
    <div class="field-desc">하나의 에피소드가 녹화될 <strong> 최대 시간(초)</strong>을 설정합니다.</div>
  </div>
  
  <div class="field-row">
    <div class="field-label">Task instruction</div>
    <div class="field-desc">로봇이 수행해야 할 구체적인 <strong> 작업 지시문</strong>을 입력합니다.</div>
  </div>
</div>

<br />

:::tip[참고]
Task Name은 실제 <strong> 데이터 폴더 경로</strong>로 사용되므로, 한글이나 특수문자, 공백 대신 <strong> 영문 소문자와 언더바(_) 사용을 권장</strong>합니다.
:::

:::tip[참고]
Task instruction은 <strong> VLA 모델 학습 시 텍스트 입력</strong>으로 활용되므로, 최대한 구체적이고 일관된 문구로 작성해 주세요.
:::

# 4.2.2 메타정보 관리
     
작성한 메타데이터는 저장 기능을 통해 효율적으로 관리할 수 있습니다.

<div class="field-table">
  <div class="field-row">
    <div class="field-label">Save</div>
    <div class="field-desc">현재 입력된 메타데이터 저장</div>
  </div>

  <div class="field-row">
    <div class="field-label">Load</div>
    <div class="field-desc">저장된 메타데이터 불러오기</div>
  </div>

  <div class="field-row">
    <div class="field-label">Delete</div>
    <div class="field-desc">저장된 메타데이터 삭제</div>
  </div>
</div>

<br />

#### 설정 저장 (Save)
- Save Preference 창에서 파일 이름을 지정하고 <strong> [Save]</strong>를 클릭합니다.
![설정 저장](./image/4.2_save_preference.gif)     
<br />

#### 설정 불러오기 및 삭제 (Load & Delete)
- 불러오기: Load Preference 창의 목록에서 원하는 파일을 선택한 후 <strong> [Load]</strong>를 클릭합니다.
![설정 불러오기](./image/4.2_load_preference.gif)
<br />

- 삭제하기: 목록에서 삭제할 파일을 선택한 후 <strong> [Delete]</strong>를 클릭합니다.
![설정 불러오기](./image/4.2_delete_preference.gif)
