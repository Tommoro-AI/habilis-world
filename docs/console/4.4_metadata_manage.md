## 상세 조작 방법
#### 설정 저장 (Save)
- Save Preference 창에서 파일 이름을 지정하고 <strong>[Save]</strong>를 클릭합니다.
![설정 저장](./image/4.2_save_preference.gif)     

#### 설정 불러오기 및 삭제 (Load & Delete)
- 불러오기: Load Preference 창의 목록에서 원하는 파일을 선택한 후 <strong>[Load]</strong>를 클릭합니다.
![설정 불러오기](./image/4.2_load_preference.gif)

- 삭제하기: 목록에서 삭제할 파일을 선택한 후 <strong>[Delete]</strong>를 클릭합니다.
![설정 불러오기](./image/4.2_delete_preference.gif)
