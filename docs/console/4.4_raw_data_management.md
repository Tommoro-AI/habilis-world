# 4.4 수집 데이터 관리

수집된 데이터의 메타 정보를 조회하고, 불필요해진 데이터를 삭제하여 관리합니다.

## Step 1. 수집 데이터 상세 정보 조회
1. 좌측 메뉴 바에서 **[Storage]** 탭을 클릭하여 데이터 변환 화면에 진입합니다.
2. **Task name 목록**에서 원하는 Task name을 클릭합니다.

![예시 이미지](./image/4.4_select_task_name.png)

3. 원하는 시간대에 수집된 데이터를 클릭하여 상세 정보를 확인할 수 있습니다.

![예시 이미지](./image/4.4_select_task_name_detail.gif)

## Step 2. 수집 데이터 삭제

- 목록에서 다시 사용하지 않거나 수집 실패로 판명된 수집 데이터 항목을 선택한 뒤, **[삭제]** 버튼을 클릭하여 삭제를 진행합니다.
![예시 이미지](./image/4.4_delete.gif)

- 삭제 전 다른 항목이 잘못 선택되었다면, **[선택 해제]** 버튼을 눌러 선택된 항목을 초기 상태로 되돌립니다.
![예시 이미지](./image/4.4_clear.gif)
