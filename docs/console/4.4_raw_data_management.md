# 5.1 원본 데이터 관리
수집된 원본 데이터를 확인하고 삭제합니다.

## Step 1. 데이터 상세 정보
상세 정보를 보고 싶은 데이터를 선택합니다.

![예시 이미지](../../public/page5/detail.gif)
:::info[Information]
- 상세 정보
    - Serial number
    - FPS
    - Duration
    - Robot Type
    - Task instruction
    - tags
    - 수집 일자: 데이터 오른쪽 상단에 있습니다.
:::
**[Refresh]** 버튼으로 최신 데이터로 새로고침 할 수 있습니다.
![예시 이미지](../../public/page5/refresh.gif)

**[Clear]** 버튼으로 선택을 초기화 할 수 있습니다.
![예시 이미지](../../public/page5/clear.gif)

## Step 2. 데이터 삭제
삭제하고 싶은 데이터를 선택 후 **[Delete]** 버튼을 클릭 합니다. 
![예시 이미지](../../public/page5/delete.gif)
