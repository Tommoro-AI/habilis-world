# 4.4 수집 데이터 관리

수집된 데이터의 메타 정보를 `조회`하고, 필요하지 않은 데이터를 `삭제`하고 lerobot 데이터 형식으로 `변환`할 수 있습니다.

## Step 1. 상세 정보 조회
1. 좌측 메뉴 바에서 [Storage] 탭을 클릭하여 데이터 변환 화면으로 이동합니다.
2. **Task name 목록**에서 확인하려는 Task를 선택합니다.

![예시 이미지](./image/4.4_select_task_name.png)
<br />

3. 선택한 Task에서 원하는 시간대에 수집된 데이터를 클릭하면 해당 데이터의 상세 정보를 확인할 수 있습니다.

![예시 이미지](./image/4.4_select_task_name_detail.gif)
<br />

## Step 2. 수집 데이터 삭제

- 더 이상 사용하지 않거나 수집에 실패한 데이터는 삭제하여 관리할 수 있습니다.
- 삭제하려는 데이터 항목을 선택한 후 **[삭제]** 버튼을 클릭하면 해당 데이터가 삭제됩니다.
- 삭제 전에 다른 항목이 잘못 선택된 경우에는 **[선택 해제]** 버튼을 클릭하여 선택 상태를 초기화할 수 있습니다.
![예시 이미지](./image/4.4_clear.gif)
<br />

## Step 3. lerobot 형식 변환 
**[변환]** : 버튼을 클릭하면 선택한 데이터가 LeRobot 형식으로 변환됩니다.<br />
변환 진행 상황은 화면 상단의 진행 상태 영역에서 실시간으로 확인할 수 있습니다.

**[취소]** : 버튼을 클릭하면 진행 중인 변환 작업을 중지할 수 있습니다.<br />
취소할 경우, 중지 시점까지 변환이 완료된 데이터만 저장됩니다.
![예시 이미지](./image/4.5_convert.gif)

<br />

:::tip[참고]
태그 기능을 활용하여 원하는 데이터만 선택적으로 변환할 수 있습니다.
- <strong> 태그 한번 클릭</strong>하면 특정 태그를 <strong> 제외하고</strong> 변환합니다.
- <strong> 태그  다시 한 번 클릭</strong>하면 특정 태그를 <strong> 포함하여</strong> 변환합니다. 
![예시 이미지](./image/4.5_select_tags.gif)
:::
<br />


## Step 4. 데이터 저장 경로   
변환이 완료된 데이터셋은 **dataset_manager가 설치된 컴퓨터**의 다음 경로에서 확인할 수 있습니다.
- 저장 경로: `~/.local/com.hablis.launcher/habilis/data/lerobot`
- 데이터 구조 예시:    
```bash
~/.local/com.hablis.launcher/habilis/data/lerobot
    └── {task_name}/                                                                 
        ├── data                                                                         
        ├── meta                                                     
        └── videos                                                       
```
