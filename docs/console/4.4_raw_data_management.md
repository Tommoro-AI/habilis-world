# 5.1 원본 데이터 관리
수집된 원본 데이터를 확인하고 삭제합니다.

## Step 1. 데이터 상세 정보
상세 정보를 보고 싶은 데이터를 선택합니다.

![예시 이미지](./image/detail.gif)
:::info[Information]
- 기본 정보
    - 테스크 이름
    - 태그
    - 생성 시간
- 상세 정보
    - 일련번호(Serial)
    - FPS
    - 재생시간
    - 로봇 모델
    - 작업 지시문
:::
**[새로고침]** 버튼으로 최신 데이터로 새로고침 할 수 있습니다.
![예시 이미지](./image/refresh.gif)

**[Clear]** 버튼으로 선택을 초기화 할 수 있습니다.
![예시 이미지](./image/clear.gif)

## Step 2. 데이터 삭제
삭제하고 싶은 데이터를 선택 후 **[Delete]** 버튼을 클릭 합니다. 
![예시 이미지](./image/delete.gif)`

**[선택 해제]** 버튼으로 선택을 초기화 할 수 있습니다.
![예시 이미지](./image/clear.gif)
