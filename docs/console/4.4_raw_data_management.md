# 4.4 원본 데이터 관리

수집된 원본 데이터의 메타 정보를 조회하고, 불필요해진 데이터를 삭제하여 관리합니다.

## Step 1. 데이터 상세 정보 조회

목록에서 특정 원본 데이터를 선택해 세부 내역을 상세히 확인합니다. 

- **조회 방법:** 리스트에서 상세 내용을 확인하고 싶은 원본 데이터(Task name)를 클릭합니다.

![예시 이미지](./image/4.4_select_task_name.png)

- **상세 정보 확인:** 클릭 후 우측 화면을 통해 원본 데이터의 구체적인 정보가 나타납니다.

:::info[Data Information]
- **기본 상태 정보**
  - 테스크 이름 (Task name)
  - 등록된 태그
  - 수집 완료 및 생성 시간
- **물리 및 기기 상세 정보**
  - 일련번호 (Serial)
  - 초당 프레임 수 (FPS)
  - 총 재생시간
  - 적용된 로봇 모델
  - 작업 지시문 (Instruction)
:::

- **데이터 목록 갱신:** **[새로고침]** 버튼을 눌러 최신 데이터 상태로 리스트를 업데이트합니다.
![예시 이미지](./image/refresh.gif)

- **조회 및 선택 초기화:** **[선택 해제]** 버튼을 통해 지정 상태를 해제합니다.
![예시 이미지](./image/clear.gif)

## Step 2. 데이터 삭제

다시 사용하지 않거나 수집 실패로 판명된 원본 데이터를 삭제합니다.

- **데이터 삭제:** 목록에서 삭제할 데이터를 선택한 뒤, **[Delete]** 버튼을 클릭하여 완전 삭제를 진행합니다.
![예시 이미지](./image/delete.gif)

- **선택 초기화:** 삭제 전 다른 항목이 잘못 선택되었다면, **[선택 해제]** 버튼을 눌러 선택된 항목을 초기 상태로 되돌립니다.
![예시 이미지](./image/clear.gif)
