# 4.5 변환 데이터 관리
변환된 데이터를 검수하고 메타 정보를 수정할 수 있으며, 데이터셋을 분할하거나 병합하여 관리할 수 있습니다.

## Step 1. lerobot 데이터 조회
원하는 Task Name을 선택하여 에피소드 단위로 데이터를 조회할 수 있습니다.
![예시 이미지](./image/4.6_play_episode.gif)
<br />

<br />

## Step 2. lerobot 데이터셋 분할  
데이터셋을 선택한 후, 분할하려는 에피소드들을 선택합니다.
이후 [분할] 버튼을 클릭하면 선택한 에피소드들을 기준으로 데이터셋 분할이 완료됩니다.
![예시 이미지](./image/4.7_split.gif)
<br />

## Step 3. 메타 데이터 수정
에피소드 선택 화면 우측 하단에 해당 데이터의 메타 정보가 표시됩니다.
태그, 품질 평가, 작업 지시문을 수정한 후 **[변경사항 저장]** 버튼을 클릭하면 수정 내용이 반영됩니다.
- tags는 입력 후 `,` 또는 `Enter` 키를 눌러야 정상적으로 구분되어 인식됩니다.
![예시 이미지](./image/4.6_edit_episode.gif)
<br />


## Step 4. lerobot 데이터셋 병합
병합하고 싶은 데이터셋들을 선택 하고 **[병합]** 버튼을 클릭하여 병합을 완료합니다.
![예시 이미지](./image/4.7_merge.gif)

:::tip[참고]
- 데이터 수정, 분할, 병합 등 작업이 진행 중인 경우, 해당 작업이 완료될 때까지 다른 기능은 비활성화됩니다.
:::
