# 4.5 수집 데이터 형식 변환
수집 데이터를 AI 학습을 위한 LeRobot 데이터셋 형식으로 변환합니다.

## Step 1. 메뉴 접속 및 대상 선택
1. 좌측 메뉴 바에서 **[Storage]** 탭을 클릭하여 데이터 변환 화면에 진입합니다.

2. **Task name 목록** 에서 변환을 원하는 Task name을 선택합니다.
![예시 이미지](./image/4.5_select_task_name.png)

:::info[정보]
![예시 이미지](./image/4.5_select_tags.gif)
태그 기능을 활용하여 원하는 데이터만 필터링하여 변환할 수 있습니다.
- <strong> Excluded tags</strong>: 특정 태그가 포함된 에피소드를 <strong> 제외하고</strong> 변환합니다.
- <strong> Included tags</strong>: 특정 태그가 포함된 에피소드만 <strong> 포함하여</strong> 변환합니다. 
:::

## Step 2. 수집 데이터 형식 변환 실행 및 취소
**[변환]** 버튼을 클릭하여 LeRobot 형식으로 변환합니다.
상단의 **진행 상태** 섹션에서 실시간 변환 진행률을 모니터링할 수 있습니다.

**[취소]** 버튼을 클릭하여 변환을 중지할 수 있습니다.
취소시 마지막까지 변환된 데이터만 변환이 마무리되어 저장됩니다.

## Step 3. 최종 데이터 저장 경로 확인  
변환이 완료된 데이터셋은 **dataset_manager가 설치된 컴퓨터**의 다음 경로에서 확인할 수 있습니다.
- 저장 경로: `~/.local/com.hablis.launcher/habilis/data/lerobot`
- Lerobot 데이터 구조 예시:    
```bash
~/.local/com.hablis.launcher/habilis/data/lerobot
    └── {task_name}/                                                                 
        ├── data                                                                         
        ├── meta                                                     
        └── videos                                                       
```
