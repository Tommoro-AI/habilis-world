# 4.6 lerobot 데이터 검수
lerobot으로 변환된 데이터를 검수하고 grade, tags, task instruction의 메타 정보를 수정합니다.

## Step 1. lerobot 데이터 검수
원하는 Task Name을 선택하면 에피소드 단위로 데이터를 검수할 수 있습니다.
![예시 이미지](./image/4.6_episode_main.png)

:::info[정보]
![예시 이미지](./image/4.6_play_episode.gif)
- **재생 컨트롤**
    - **[play]**: 에피소드를 재생합니다.
    - **[stop]**: 에피소드를 일시정지합니다.
    - **[1x, 2x, 3x]**: 재생 속도를 조절합니다.
:::

## Step 2. lerobot 데이터 수정
Episode를 선택하면 상단엔 녹화 영상, 좌측엔 에피소드 목록, 우측엔 메타정보가 표시되며 tags, grade, task instruction을 수정한 후 **[Save Episode]** 버튼을 클릭하여 수정이 완료됩니다.
![예시 이미지](./image/4.6_edit_episode.gif)

:::info[정보]
![예시 이미지](./image/4.6_label_split.gif)
- 상단의 **[good, bad, unlabel]**을 클릭하여 원하는 grade로 필터링하여 에피소드를 조회할 수 있습니다.
- 상단의 **tags**를 이용하여 원하는 태그로 필터링하여 에피소드를 조회할 수 있습니다.
:::
