# 6.2 데이터 라벨링
데이터를 시각화하여 grade, tags, task instruction을 수정합니다.

## Step 1. 데이터 시각화
원하는 Task Name을 선택하면 데이터를 시각화하는 창이 나옵니다.
![예시 이미지](../../public/page6/label_main.png)

- **[play]**: 데이터를 재생합니다.
- **[stop]**: 데이터를 일시정지합니다.
- **[1x, 2x, 3x]**: 재생 속도를 조절합니다.
![예시 이미지](../../public/page6/play_episode.gif)

## Step 2. 데이터 라벨링
Episode를 선택하면 우측에 상세내역이 표시되며, 수정을 하고 **[Save Episode]** 버튼을 클릭하면 수정이 완료됩니다.
![예시 이미지](../../public/page6/label_detail.gif)

## Step 3. 데이터 분할해서 보기
상단의 **[good, bad, unlabel]** 혹은 tags를 이용하여 에피소드를 분할해서 볼 수 있습니다.
![예시 이미지](../../public/page6/label_split.gif)
