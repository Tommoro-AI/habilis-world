# 4.7 lerobot 데이터 수정
데이터를 grade, tags를 기준으로 분할 삭제, 병합하며 lerobot dataset v2.1로 변환할 수 있습니다.

## Step 1. lerobot 데이터 삭제
부분적으로 혹은 전체적으로 데이터를 삭제합니다.
### 1. lerobot 데이터 전체 삭제
첫 페이지에서 삭제하고 싶은 task name을 선택 후 **[삭제]** 버튼을 클릭하여 전체 삭제를 완료합니다.
![예시 이미지](./image/4.7_delete_all.gif)

### 2. lerobot 데이터 에피소드 단위 삭제
Task Name을 선택한 후 페이지에서 삭제하고 싶은 에피소드를 선택 하고 **[삭제]** 버튼을 클릭하여 에피소드 단위로 삭제를 완료합니다.
![예시 이미지](./image/4.7_delete_partial.gif)

## Step 2. lerobot 데이터 분할  
Task Name을 선택한 후 분할하고 싶은 에피소드를 선택 하고 **[분할]** 버튼을 클릭하여 분할을 완료합니다.
![예시 이미지](./image/4.7_split.gif)

:::info[Information]
![예시 이미지](./image/4.7_label_split.gif)
- 상단의 **[good, bad, unlabel]**을 클릭하여 원하는 grade로 필터링하여 에피소드를 조회할 수 있습니다.
- 상단의 **tags**를 이용하여 원하는 태그로 필터링하여 에피소드를 조회할 수 있습니다.
:::

## Step 3. lerobot 데이터 병합
병합하고 싶은 task name을 선택 하고 **[병합]** 버튼을 클릭하여 병합을 완료합니다.
![예시 이미지](./image/4.7_merge.gif)
