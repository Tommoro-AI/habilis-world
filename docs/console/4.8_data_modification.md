# 6.2 데이터 수정
데이터를 grade, tags를 기준으로 분할 삭제, 병합하며 lerobot dataset v2.1로 변환할 수 있습니다.

## Step 1. 데이터 삭제
데이터를 부분적으로 혹은 전체를 삭제합니다.
### 1. 데이터 전체 삭제
첫 페이지에서 **[Delete]** 버튼을 클릭합니다.
![예시 이미지](../../public/page6/delete_all.gif)

### 2. 데이터 부분 삭제
Task Name을 선택한 후 삭제하고 싶은 에피소드를 선택 후 **[Delete]** 버튼을 클릭 합니다.
![예시 이미지](../../public/page6/delete_partial.gif)

## Step 2. 데이터 분할
Task Name을 선택한 후 분할하고 싶은 에피소드를 선택 후 **[Split]** 버튼을 클릭 합니다.
![예시 이미지](../../public/page6/split.gif)

## Step 3. 데이터 병합
Task Name을 선택한 후 병합하고 싶은 에피소드를 선택 후 **[Merge]** 버튼을 클릭 합니다.
![예시 이미지](../../public/page6/merge.gif)
