# 5.1. 학습 데이터 선택
모델 학습에 사용할 데이터 셋을 선택합니다

## Step 1. 데이터 리스트 확인 및 선택
1. AI Manager가 인식한 데이터의 리스트틀 확인할 수 있습니다
   - AI Manager는 Lerobot 형식의 데이터셋을 기본으로 지원합니다
   - Lerobot 형식에 맞지 않는 데이터는 학습이 진행되지 않을 수 있습니다
  2. 데이터 항목을 클릭하는 것으로 학습에 사용할 데이터를 선택할 수 있습니다

![예시 이미지](./image/그림35.png)
    
## Step 2. 데이터 추가
1. AI Manager는 /-데이터 디렉토리-/에 들어있는 데이터를 인식합니다
2. 학습에 활용하고자 하는 데이터를 /-데이터 디렉토리에-/ 추가하는 것으로 데이터 리스트에 포함시킬 수 있습니다
3. 데이터를 추가한 후에, Refresh Datasets 버튼을 클릭하여 콘솔에서 새롭게 추가된 데이터를 확인할 수 있습니다

![예시 이미지](./image/그림36.png)
