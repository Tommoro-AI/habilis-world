# 5.2 학습 모델 선택

## Step 1. 학습 모델 선택
- AI Manager를 통해 학습할 수 있는 모델의 리스트를 확인할 수 있습니다
- 해당 모델을 클릭하는 것으로 학습에 사용할 모델을 선택할 수 있습니다
- 항목에 없는 모델들은 추후 업데이트를 통해 추가될 예정입니다(GR00T N1.5, Pi05 등)

![예시 이미지](./image/그림35.png)
    
## Step 2. 학습 디바이스 선택
1. 학습에 사용할 디바이스를 선택합니다
2. GPU 옵션을 선택할 경우, AI Manager가 설치되어 있는 PC의 GPU 환경이 유효한지 확인하세요
   - 예를 들어, GPU의 메모리 크기가 학습에 필요한 양보다 작은 경우 학습의 진행이 불가할 수 있습니다

![예시 이미지](./image/그림36.png)
