# 5.3 저장 폴더 이름 설정
체크포인트를 저장할 폴더의 이름을 설정합니다

## Step 1. 저장 폴더 이름 입력
- 학습의 결과인 체크포인트를 저장할 폴더의 이름을 입력합니다

![예시 이미지](./image/그림35.png)
    
## Step 2. 중복 확인
- 폴더 이름을 입력한 후, Check Duplicate 버튼을 반드시 클릭해서 중복 여부를 확인합니다
- 중복 여부를 체크하지 않으면 학습이 실행되지 않습니다
- 폴더 이름이 중복일 경우, 다른 이름으로 변경하신 후 다시 중복 체크를 진행합니다

![예시 이미지](./image/그림36.png)
