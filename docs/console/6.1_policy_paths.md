# 6.1 모델 체크포인트 선택

## 체크 포인트 선택
- Policy Paths에서 Choose policy 항목을 통해서 파일 탐색기를 열어볼 수 있습니다
- 파일 탐색기를 통해서 추론에 사용하고자 하는 체크포인트를 선택합니다
  - 선택된 체크포인트가 유효하지 않은 경우 추론이 진행되지 않을 수 있습니다
    - 하빌리스 콘솔에서 지원하지 않는 모델
    - AI Manager가 인식하지 못하는 metadata format
  - 하빌리스 콘솔을 이용해서 학습한 체크포인트를 이용하여 추론하는 것을 추천합니다
