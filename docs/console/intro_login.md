# Habilis Console 소개 및 로그인

## Habilis Console

Habilis Console은 로봇 운영 및 데이터 관리를 위한 통합 웹 플랫폼입니다. 사용자는 이 콘솔을 통해 로봇의 상태를 실시간으로 모니터링하고, 수집된 데이터를 효율적으로 확인 및 관리할 수 있습니다.

---

## 1. 로그인

1. **Habilis Console** 웹사이트에 접속합니다. 
    
    **웹사이트 접속:** 지정된 Habilis Console 공식 URL을 통해 접속합니다.
    
    [habilis-console.com](https://habilis-console.com/)
    
    ![예시 이미지](/images/example.png)
    
2. 바로 메인화면으로 진입합니다.
