# 하빌리스 콘솔이란? 
<br />

![habilis console](https://tommoro-ai.github.io/doc_habilis_console/image/Habilis_console_block_diagram.png)

<br />

---

<span style="color:#234080;">1. <strong>사용자</strong></span> :
사용자는 부여받은 계정을 통해 콘솔 웹 브라우저에 접속하여 시스템을 이용합니다.<br />

<span style="color:#234080;">2. <strong>콘솔 웹 브라우저</strong></span> :
로봇 제어, 데이터 수집, 학습 및 추론 등 콘솔의 주요 기능을 제공하는 웹 기반 인터페이스입니다.
<a href="https://habilis-console.com/"
   target="_blank"
   rel="noopener noreferrer"
   style="color:#1B4F9C; font-weight:600;">
   링크
</a><br />


<span style="color:#234080;">3. <strong>Habilis Robot Manager</strong></span> :
하빌리스 콘솔 사용을 위해 필수적으로 설치되어야 하는 패키지로 로봇과의 통신을 담당하며, 원격 제어 및 상태 관리 기능을 제공합니다.
(ex: rby1a, rby1m 등) <strong style="color:#B03A2E;">(필수)</strong><br />

<span style="color:#234080;">4. <strong>Habilis Control Retargeter</strong></span> :
VR Manager 등 추가 입력 제어 장치와의 연동을 지원하여
다양한 방식의 로봇 조작 환경을 제공합니다.<br />


<span style="color:#234080;">5. <strong>Habilis Dataset Manager</strong></span> :
로봇의 동작 데이터 및 시각 데이터를 수집하고,
이를 학습 가능한 데이터셋 형태로 변환·관리하는 패키지입니다.<br />

<span style="color:#234080;">6. <strong>Habilis Data Storage</strong></span> :
대규모 데이터 저장을 위한 추가 스토리지를 제공합니다.
<strong style="color:#B03A2E;">(별도 문의)</strong><br />


<span style="color:#234080;">7. <strong>Habilis AI Manager</strong></span> :
수집된 로봇 데이터를 기반으로 인공지능 모델을 학습하고,
실시간 추론 및 모델 관리 기능을 제공합니다.<br />
