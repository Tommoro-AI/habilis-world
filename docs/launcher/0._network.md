# 로봇실 사전 주의사항

### 1. 네트워크
반드시 [공유기 1대 - 로봇 1대 - PC 1대] 세트로 연결해 주세요.
![Network Setting](./image/warning_network.png)

<div style="color:#2271b1">추후 업데이트를 통해 하나의 라우터 내부에 여러대의 로봇, PC가 지원될 예정입니다.</div>

-----------------------------

### 2. 카메라
현재 데이터 수집시 오류를 방지하기 위해 특정 카메라가 연결 되었을 시에만 로봇이 켜지도록 되어있습니다.
- Realsense D405 x2(wrist_right, left cam)
- Zed(zed 2i, zedm ...)x1(head cam)

카메라와 로봇을 연결 후 로봇을 사용해주시기 바랍니다.
![Camera Setting](./image/warning_camera.png)

<div style="color:#2271b1">추후 업데이트를 통해 다양한 카메라가 지원될 예정입니다.</div>
