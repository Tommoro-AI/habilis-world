# 0. 하빌리스 콘솔 주의사항

## 1. 네트워크 주의사항
하나의 라우터에 한대의 로봇과 PC(확장팩 1종류당 하나씩)를 사용해주시기 바랍니다
![Network Setting](./image/0_network_caution.png)

-<strong>추후 업데이트로 하나의 라우터 내부에 여러대의 로봇, PC가 지원될 예정입니다.</strong>

## 2. 로봇 주의사항
현재 데이터 수집시 오류를 방지하기 위해 특정 카메라가 연결 되었을 시에만 로봇이 켜지도록 되어있습니다.
- Realsense D405 x2(wrist_right, left cam)
- Zed(zed 2i, zedm ...)x1(head cam)

카메라와 로봇을 연결 후 로봇을 사용해주시기 바랍니다.
![Camera Setting](./image/0_robot_camera_caution.png)

-<strong>추후 업데이트로 다양한 카메라가 지원될 예정입니다.</strong>
