# 1.1 설치 파일 다운로드

본인의 운영체제(OS) 환경에 맞는 설치 파일을 선택하여 다운로드하세요.
설치 파일은 
<a href="https://tommoro.ai/download"
   target="_blank"
   rel="noopener noreferrer"
   style="color:#1B4F9C; font-weight:600;">
   다운로드 페이지
</a> 에서 받을 수 있습니다.

![설치 파일 다운로드](./image/1_launcher_download_page.png)


:::tip[참고]
리눅스의 경우 다음의 명령어로 CPU 아키텍처를 확인한 뒤 본인의 환경에 적합한 버전으로 설치해주시면 됩니다.
```bash
uname -m | sed 's/aarch64/arm64/; s/x86_64/amd64/'
```

출력
```bash 
amd64 #AMD 버전 설치
OR
arm64 #ARM 버전 설치 
```

:::
