# 1.2 윈도우 설치 방법

윈도우 환경에서는 설치 마법사(MSI)를 통해 간편하게 설치할 수 있습니다.

## Step 1. 설치

1. 다운로드한 설치 파일(`.msi`)을 찾아 더블 클릭하여 실행합니다.
![예시 이미지](./image/example.png)
    
2. 설치 창이 열리면 안내에 따라 [Next] 버튼을 클릭하고, 이어서 <strong>[Install]</strong>을 눌러 설치를 시작합니다.
![예시 이미지](./image/example.png)

:::tip[참고]
보안 경고 창(SmartScreen)이 나타나면 추가 정보 클릭 후 실행 버튼을 누르세요.
:::

3. 진행바가 모두 채워지면 설치가 완료된 것입니다. <strong>[Finish]</strong> 버튼을 클릭하여 설치 마법사를 닫아주세요.
![예시 이미지](./image/example.png)
    
## Step 2. 하빌리스 런처 실행

1. 바탕화면에 생성된 '하빌리스 런처' 아이콘을 실행하거나, 윈도우 검색창에서 이름을 검색하여 프로그램을 실행하세요.
![예시 이미지](./image/example.png)
