
## 1.3 리눅스 설치 방법

리눅스 환경에서는 AppImage 실행 파일을 실행하여 하빌리스 런처를 시작합니다.  
**하빌리스 런처 실행을 위해 Podman이 사전에 설치되어 있어야 합니다.**

### 1.3.1 설치 파일 다운로드

1. 다운로드 페이지에서 `.AppImage` 파일을 받습니다.
2. 기본적으로 파일은 **다운로드(Downloads) 폴더**에 저장됩니다.

![설치 파일 다운로드](./image/download_page.png)

### 1.3.2 Podman 설치 (필수)

하빌리스 런처는 내부적으로 컨테이너 환경을 사용하므로 **Podman 설치가 필요합니다.**  
사용 중인 리눅스 배포판에 맞게 아래 명령어를 실행해 주세요.

#### Ubuntu / Debian 계열
```bash
sudo apt update
sudo apt install -y podman
````

#### Fedora

```bash
sudo dnf install -y podman
```

#### CentOS / Rocky 리눅스 / Alma리눅스

```bash
sudo dnf install -y podman
```

설치 후 정상적으로 설치되었는지 확인합니다.

```bash
podman --version
```

### 1.3.3 실행

1. 터미널에서 **다운로드 폴더로 이동**합니다.

```bash
cd ~/Downloads
```

2. 파일 실행 권한을 부여합니다.

```bash
chmod +x habilis-launcher.AppImage
```

3. AppImage 실행

```bash
./habilis-launcher.AppImage
```

:::info[정보]

* `chmod +x` 명령은 **AppImage 파일이 있는 폴더**에서 실행해야 합니다.
* AppImage 실행 시 오류 메시지가 나온다면 `chmod +x habilis-launcher.AppImage` 를 다시 시도해주세요.
* Podman이 설치되어 있지 않으면 Launcher가 정상적으로 동작하지 않습니다.
:::
