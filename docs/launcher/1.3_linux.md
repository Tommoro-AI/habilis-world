# 1.3 리눅스 설치 방법

리눅스 환경에서는 AppImage 실행 파일을 통해 하빌리스 런처를 실행할 수 있습니다.  
아래 절차에 따라 파일을 다운로드하고 실행하십시오.

---

## Step 1. 다운로드

1. 파일을 다운로드할 디렉토리로 이동합니다.

```bash
cd ~/Downloads
```

2. 최신 릴리스를 다운로드합니다.

    아래 명령어 중 하나를 선택하여 실행하십시오.

**<span style="color:#1E90FF;">방법 A</span> — curl 사용**

```bash
curl -L -o habilis-launcher.AppImage https://dl.habilis-console.com/launcher/latest/linux-amd64-appimage/download
```

**<span style="color:#1E90FF;">방법 B</span> — wget 사용**

```bash
wget -O habilis-launcher.AppImage https://dl.habilis-console.com/launcher/latest/linux-amd64-appimage/download
```
**<span style="color:#1E90FF;">방법 C</span> — 홈페이지 다운로드 페이지 이용**


<br />

3. 다운로드한 파일에 실행 권한을 부여합니다.

```bash
chmod +x habilis-launcher.AppImage
```
<br />

> ⚠️ **주의**
> - `chmod +x` 명령은 AppImage 파일이 위치한 디렉토리에서 실행해야 합니다.
> - 또는 파일 탐색기에서 해당 파일을 마우스 오른쪽 클릭 → **속성** → **권한** 탭 →  
>   <strong>"프로그램으로 실행 허용"</strong>을 체크하여 동일하게 설정할 수 있습니다.

<br />

## Step 2. 하빌리스 런처 실행

아래 명령어를 실행하여 런처를 시작합니다.

```bash
./habilis-launcher.AppImage
```

정상적으로 실행되면 하빌리스 런처가 시작됩니다.
