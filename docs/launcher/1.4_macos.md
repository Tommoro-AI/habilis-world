# 1.4 맥OS 설치 방법 (비공개 페이지)
맥OS 환경에서는 .app 파일을 Applications 폴더로 옮겨 설치합니다.

## Step 1. 설치

1. 다운로드한 `habilis-launcher.app` 파일을 확인합니다.
![예시 이미지](./image/1.4_(1).png)

2. 해당 앱을 응용 프로그램(Applications) 폴더로 드래그하여 이동합니다.
![예시 이미지](./image/1.4_(2).png)


## Step 2. 보안 설정 (최초 1회)
맥의 보안 정책상 공식 앱스토어 외의 앱은 실행이 차단될 수 있습니다. 아래 절차를 통해 보안 속성을 해제해 주세요.

1. 터미널(Terminal) 을 실행합니다.
![예시 이미지](./image/1.4_(3).png)

2. 아래 명령어를 복사하여 입력한 뒤 엔터(Enter)를 누릅니다.
```​bash
xattr -cr /Applications/habilis-launcher.app
```
![예시 이미지](./image/1.4_(4).png)

## Step 3. 하빌리스 런처 실행
1. 응용 프로그램 폴더에서 하빌리스 런처를 실행할 수 있습니다.
![예시 이미지](./image/1.4_(5).png)

:::warning[경고]
![예시 이미지](./image/1.4_(6).png)
**"앱이 손상되었기 때문에 열 수 없습니다"** 또는 **"확인되지 않은 개발자"** 등의 메시지가 표시된다면
위의 xattr -cr 명령이 적용되지 않았거나, 앱이 업데이트/재설치로 교체된 경우일 수 있습니다.
이 경우 터미널 명령을 다시 실행한 후 앱을 실행하세요.
:::
