## 2. 로그인

하빌리스 런처 사용을 위해서는 로그인이 필요하며, 로그인 후 전체 메뉴에 접근할 수 있습니다.

## Step 1. 계정 정보 확인

- 로그인에 필요한 이메일과 임시 비밀번호는 **회사 인보이스에 첨부된 안내 메일**에서 확인할 수 있습니다. 
메일함에서 해당 내용을 먼저 확인해 주세요.
<br />
- 계정 비밀번호를 잊으셨나요? https://habilis-console.com/ 홈페이지에서 재발급 신청하시기 바랍니다. 


## Step 2. 로그인 진행

1. 로그인 화면에서 **Email / Password**를 입력합니다.
![런처 로그인 화면](./image/launcher_login_1.png)


2. 입력 후 하단의 **[로그인]** 버튼을 클릭합니다.
<br />

:::warning[경고]
<strong> 의존성 문제</strong>
하빌리스 콘솔에 필요한 라이브러리들이 존재하지 않을 경우 설치 권한을 요청합니다. 

![의존성 화면](./image/requirements.png)

노트북 계정 비밀번호 입력 후 설치가 진행 됩니다.



![의존성 화면2](./image/requirements_2.png)

설치가 완료되면 Habilis communicator가 실행되고 런처 사용이 가능해집니다.

![의존성 화면2](./image/requirements_3.png)

:::



> <strong> 임시 비밀번호 변경</strong>
>
> 최초 로그인 시 제공된 임시 비밀번호는 보안을 위해 반드시 변경해야 합니다.
> 로그인 후 [설정 > 계정 관리] 메뉴를 통해 본인만의 비밀번호로 업데이트해 주세요.
