## 2. 로그인

Habilis Launcher 사용을 위해서는 로그인이 필요하며, 로그인 후 전체 메뉴에 접근할 수 있습니다.

1. 로그인 화면에서 **Email / Password**를 입력합니다.

![런처 로그인 화면](./image/launcher_login.png)


2. 로그인 정보는 **회사 인보이스**에 첨부된 안내 메일(첨부 내용)에 포함된 이메일과 비밀번호(임시 비밀번호)를 사용합니다.

3. 입력 후 **로그인** 버튼을 클릭합니다.


> **[ 참고 ] 로그인 정보는 어디서 확인하나요?**
>
> - 계정 정보(이메일/비밀번호)는 **회사 인보이스에 첨부된 안내 메일(첨부 내용)**에서 확인할 수 있습니다.


> **[ 중요 ] 임시 비밀번호 변경 안내**
>
> - 최초 로그인 후 제공된 **임시 비밀번호는 반드시 변경**해야 합니다.
> - 임시 비밀번호를 그대로 사용할 경우 보안상 문제가 발생할 수 있으니, 최초 로그인 후 비밀번호 변경을 권장합니다.
