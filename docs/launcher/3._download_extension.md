##  3. 패키지 다운로드

하빌리스 런처는 로봇 데이터 수집 및 관리 환경 구성하는 필수팩과 다양한 확장팩을 제공합니다.

### Step 1. 필수팩 설치

1. 로그인 완료 후, 상단 메뉴에서 **[필수]** 버튼을 클릭합니다.  

2. 로봇 제어에 필수적인 패키지를 설치합니다. 현재 가지고 있는 로봇 기종에 맞춰 UPC(로봇PC)안에 설치합니다.

3. ROS ID를 [apply]하지 않으면 기본 값인 `ROS_DOMAIN_ID=30`으로 설정됩니다. 

![필수 화면](./image/required_package.png)
<br />



### Step 2. 확장팩 설치


1. 상단 메뉴에서 **[확장팩]** 버튼을 클릭합니다.  
2. download 버튼 클릭 시, 로봇 데이터 학습을 비롯한 여러가지 하빌리스 콘솔의 기능들을 다운받을 수 있습니다. 

![확장팩 다운로드](./image/extension_package.png)
<br />


### Step 3. 확장팩 삭제
1. delete 버튼을 클릭 시, 다운받은 패키지를 삭제 할 수 있습니다. 

![확장팩 다운로드](./image/package_delete.png)
