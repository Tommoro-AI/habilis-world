# 확장팩 관리

## 3. 확장팩 다운로드

Habilis Launcher는 필요한 기능을 **확장팩(Extension Package)** 형태로 제공합니다.

> 확장팩은 로봇 데이터 수집 / 관리 / 실행 환경 구성을 돕는 모듈로, **Store** 메뉴에서 다운로드 후 실행 가능합니다.

---

### 3.1 Store 화면 접속

1. 로그인 완료 후, 상단 메뉴에서 **Store** 버튼을 클릭합니다.
    
    [이미지]
    
2. Store 화면에서 다운로드 가능한 확장팩 목록이 표시됩니다.

[이미지]

---

### 3.2 확장팩 검색

- 상단의 **Search(press Enter)** 입력창에서 원하는 확장팩 이름을 검색합니다.
    
    [이미지]
    

---

### 3.3 확장팩 다운로드

확장팩 카드 우측 아이콘을 통해 다운로드 상태를 확인할 수 있습니다.

1. 우측의 **다운로드 아이콘(⬇️)** 을 누르면 다운로드가 시작됩니다.
    
    [이미지]
    
2. 다운로드가 완료되면 **체크 아이콘(✅)** 으로 표시됩니다.
    
    [이미지]
    
3. 체크 표시된 확장팩은 추가 다운로드 없이 바로 사용할 수 있습니다.
    
    [이미지]
    

> **⚠️ [ 주의 ] CPU 아키텍처 호환성**
>
> 확장팩은 실행 중인 호스트 PC의 CPU 아키텍처(예: `amd64`, `arm64`)에 따라 호환 여부가 달라질 수 있습니다.
> 확장팩 목록에 **“현재 아키텍처와 다릅니다.”**라는 문구가 표시될 경우, 현재 사용 중인 PC 환경과 해당 확장팩의 아키텍처가 일치하지 않음을 의미합니다.
> 아키텍처가 맞지 않는 확장팩은 **다운로드가 완료되더라도 정상적으로 실행되지 않을 수 있으므로**, 반드시 사용 중인 환경과 호환되는 확장팩인지 확인한 후 설치를 진행해 주시기 바랍니다.

---

## 4. 확장팩 실행

Library 메뉴에서는 설치된 확장팩의 목록을 확인할 수 있으며, 필요에 따라 **실행 또는 삭제**가 가능합니다.

---

### 4.1 Library 메뉴 접속

1. 런처 상단 메뉴에서 **Library** 탭을 클릭합니다.
    
    [이미지]
    
2. Store에서 다운로드한 확장팩 목록이 표시됩니다. (예: `python`, `wuji-hand`)
    
    [이미지]
    

---

### 4.2 확장팩 실행

1. 실행하려는 확장팩 항목 우측의 **실행(▶)** 아이콘을 클릭합니다.
    
    [이미지]
    
2. 확장팩이 정상적으로 실행되면 해당 아이콘이 **초록색 테두리**로 표시됩니다.
    
    [이미지]
    
    > **📖 [ 참고 ]**
    > 초록색 테두리는 해당 확장팩이 현재 실행 중(활성화)임을 의미합니다.

---

## 5. 확장팩 삭제

확장팩을 잘못 설치했거나 버전 정리가 필요한 경우, 확장팩을 삭제할 수 있습니다.

1. 우측의 **삭제 버튼**을 클릭합니다.
    
    [이미지]
    
2. 삭제된 확장팩은 목록에서 사라지며, 필요 시 **Store**에서 다시 다운로드할 수 있습니다.
    
    > **⚠️ [ 주의 ]**
    > 실행 중인 확장팩을 삭제하면 기능 오류가 발생할 수 있습니다. 삭제 전, 반드시 실행을 종료해주세요.
